# 数律智检 - 使用说明

## 首次运行配置

1. **复制并填写配置文件**
   ```
   将 .env.example 复制为 .env，并用记事本填写以下内容：
   DEEPSEEK_API_KEY=你的DeepSeek密钥
   JWT_SECRET_KEY=任意随机字符串（如 abc123xyz456）
   ```

2. **双击运行**
   ```
   数律智检.exe
   ```
   首次运行将自动弹出 API Key 配置界面。

## 系统要求

- Windows 10 / 11 (64位)
- 需要本机已安装 Python 3.11+（附带完整依赖）

> **如尚未安装依赖**，请在本目录打开命令提示符运行：
> ```
> pip install fastapi uvicorn chromadb sentence-transformers llama-index-core llama-index-vector-stores-chroma llama-index-embeddings-huggingface openai python-dotenv python-multipart psutil pywebview
> ```
