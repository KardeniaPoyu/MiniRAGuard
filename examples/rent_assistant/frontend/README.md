# MiniRAGuard Frontend | MiniRAGuard前端

基于 Uni-app (Vue) 开发的“MiniRAGuard AI”微信小程序客户端。

## ✨ 界面预览

- **首页**：合同拍照上传入口。
- **审查页**：高亮显示风险条款，分级解析法律依据。
- **咨询页**：沉浸式 AI 对话，支持上下文关联。
- **规则库**：常用法律知识普法展示。

## 🛠️ 开发准备

1.  **安装编辑器**：推荐使用 [HBuilderX](https://www.dcloud.io/hbuilderx.html)。
2.  **配置后端地址**：
    - 修改 `config.js` 中的 `BASE_URL` 为你的后端服务地址。

## 🚀 启动与编译

### 在 HBuilderX 中运行：
1.  点击 `文件 -> 导入 -> 从本地目录导入`，选择本项目。
2.  点击顶部菜单 `运行 -> 运行到小程序模拟器 -> 微信开发者工具`。

### 生产打包：
1.  点击顶部菜单 `发行 -> 小程序-微信 (仅适用于uni-app)`。
2.  在弹窗中填写 AppID，点击发行。

## 🎨 视觉规范

项目采用深色商务法律风格：
- 主色调：`#1A3A5C` (Navy Blue)
- 背景色：`#F4F6F9`
- 成功色/低风险：`#27AE60`
- 警告色/中风险：`#F39C12`
- 错误色/高风险：`#E74C3C`
